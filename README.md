# js算法集合
此库用于个人学习算法
### 使用:
```
npm install -D merlion

let fnc = require('merlion');

fnc.bubbleSort([4, 2, 1, 0.5, -3, 13]);
```

### 现支持算法
|-- 排序<br>
|---- 冒泡排序（bubbleSort）<br>
|---- 选择排序（selectionSort）<br>
|---- 插入排序（insertionSort）<br>
|---- 希尔排序（shellSort）<br>
|---- 归并排序（mergeSort）<br>
|---- 快速排序（quickSort）<br>
|---- 堆排序（heapSort）<br>
|---- 计数排序（countingSort）<br>
|---- 桶排序（bucketSort）<br>
|---- 基数排序（radixSort）<br>
